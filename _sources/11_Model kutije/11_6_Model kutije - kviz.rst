.. quizq::

    .. fillintheblank:: model_kutije_kviz_2

        Допуни наредни CSS код тако да се ширина елемента *div* постави на 200 пиксела:

        | div {
        |   |blank|: 200px ;
        | }

        - :^width$: Одговор је тачан
          :x: Одговор није тачан

.. quizq::

    .. fillintheblank:: model_kutije_kviz_3

        Допуни наредни CSS код тако да све унутрашње маргине елемента *p* поставе на 20 пиксела:

        | p {
        |   |blank|: |blank| ;
        | }

        - :^padding$: Први одговор је тачан
          :x: Први одговор није тачан
        - :^(20px)|(20px\s+20px)|(20px\s+20px\s20px\s+20px)$: Други одговор је тачан
          :x: Други одговор није тачан
